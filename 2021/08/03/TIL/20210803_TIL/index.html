💡 오늘 한 일

  • 파이썬 인강 시청

💬 느낀 점

  • 업무 자동화를 위한 수업을 듣고 있는데 파이썬에서 패키지 관리자 활용하기, 텍스트 파일 활용하기, 엑셀 파일 활용하기 등을 배웠다.

  • 앞으로 이메일을 활용하는 법을 배울 것이 남았는데 이것을 활용해서 엑셀 DB를 이용한 이메일 발송 자동화 프로그램을 구현할 것이다. 사용자가 원하는 키워드를 입력 받아서 네이버 뉴스를 수집해주는 모듈을 이용해서 해당 키워드 뉴스 수집 후 엑셀 파일에 제목, 링크, 요약문을 기록하고 그 수집 데이터 엑셀 파일을 메일링 대상자 엑셀 파일을 읽어 대상자들에게 메일 보내는 프로그램을 구현할 것이다.

    Quiz ✍🏻

    1. 다음 중 __name__ 내장 변수 설명 중 틀린 것은 ?

    1. 파이썬에서 가지고 있는 기본 내장 변수이다.
    2. 테스트 할때 사용을 한다.
    3. __name__ 변수를 가지고 있는 파일을 직접 실행시키면 __main__ 이 출력된다
    4. from ~ import 에 대해서 실행된다
    답/해설 💯

    답 : 4번

    __name__ 테스트 할 때 사용되며, 변수를 가지고 있는 파일을 실행시키면 __main__ 이 출력됩니다.

    방식으로 사용됩니다.

    from ~ import 에 대해서는 실행이 되지 않습니다.

    2. 다음 중 CSV 파일의 설명으로 틀린 것은?

    1. comma-seperated values의 약자이다.
    2. , 와 개행 문자로 구분된 파일이다.
    3. 엑셀과 유사한 형태로 읽고 쓸 수 있다.
    4. , 로 행(row)을 구분하고 개행 문자로 열(col)을 구분합니다.
    답/해설 💯

    답 : 4번

    , 로 열(column)을 구분하고 개행 문자로 행(row)을 구분합니다.

    3. 다음 설명에 해당하는 것으로 맞는 것은 ?

    1. read_only=True
    2. max_col
    3. Workbook
    4. create_sheet()
    답/해설 💯

    답 : 1번
