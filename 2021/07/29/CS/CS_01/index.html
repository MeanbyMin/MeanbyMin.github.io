[운영체제] 운영체제와 운영체제의 역할

작성일 2021-07-29 Edited on 2021-07-30 In CS Views: Disqus:

💡 운영체제란 ?

운영체제(Operating System) 는 사용자가 컴퓨터를 사용하기 위해 필요한 소프트웨어이다. 우리가 일반적으로 컴퓨터를 사용하면서 실행한 모든 프로그램들은 운영체제에서 관리하고 제어한다.

대표적인 운영체제는 Windows OS, Unix, Mac OS 등이 있다.

📌 운영체제의 역할

  1. 시스템 자원(System Resource) 관리자
  2. 사용자와 컴퓨터간의 커뮤니케이션 지원
  3. 컴퓨터 하드웨어와 프로그램을 제어

시스템 자원(System Resource) 관리자

  • 시스템 자원은 컴퓨터 하드웨어를 뜻한다.
    • CPU(중앙처리장치), Memory(DRAM, RAM)
    • I/O Device (입/출력 장치)
      • 모니터, 마우스, 키보드, 네트워크
    • 저장매체 : SSD, HDD(하드디스크)

컴퓨터 하드웨어는 스스로 할 수 있는 것이 없다.

  1. CPU : 각 프로그램이 얼마나 CPU를 사용할 지 결정할 수 없다.
  2. Memory : 각 프로그램이 어느 주소에 저장되어야 하는지, 어느 정도의 메모리 공간을 확보해줘야 하는지 결정할 수 없다.
  3. 저장매체(HDD, SDD) : 어떻게, 어디에 저장할지 결정할 수 없다.
  4. 키보드/ 마우스 : 스스로 화면에 표시할 수 없다.

위의 모든 행동을 결정하는 것이 운영체제이다. 즉, 컴퓨터의 하드웨어를 관리하는 소프트웨어이다.

사용자와 컴퓨터 간의 커뮤니케이션 지원

응용프로그램이란 ?

  • 프로그램은 소프트웨어를 의미한다.
  • 소프트웨어는 운영체제와 응용프로그램으로 나뉜다.
  • 응용프로그램은 Application(PC에서 사용하는 프로그램), App(폰에서 사용하는 프로그램)으로 불린다.

우리가 주로 사용하는 엑셀, 포토샵, 일러스트, 한글, 파워포인트가 응용프로그램이다.

운영체제와 응용 프로그램간의 관계

  • 운영체제는 응용 프로그램을 관리한다.
    • 운영체제가 응용 프로그램을 실행시킨다.
    • 응용 프로그램간의 권한을 관리해준다.
      • 관리자 권한으로 실행한다.
    • 응용 프로그램을 사용하는 사용자도 관리한다. (로그인을 사용)

컴퓨터 하드웨어와 프로그램을 제어

응용 프로그램은 누구나 만들 수 있기 때문에, 그것을 운영체제가 제어를 해준다.

  • 응용 프로그램에서 무한 루프가 돌고 있다면 운영체제가 종료를 시켜주기도 한다.
  • 응용 프로그램이 에러가 났을 경우에는 운영체제가 프로그램을 다운시키기도 한다.
  • 모든 파일 삭제를 막아준다.(권한/사용자 관리를 통해)
  • 응용 프로그램의 비정상적인 작동을 운영체제가 막는다.

🚨 운영체제의 목표

사용자가 사용하는 응용 프로그램이 효율적으로, 적절하게 동작하도록 지원한다.

운영체제는 저장매체(SSD/HDD)에 저장(설치)이 된다.

컴퓨터를 켜면 운영체제는 Memory에 올라가게 되는데, 폰 노이만 구조를 따른다.

출처 : https://m.blog.naver.com/with_msip/221981730449

  • 폰노이만 구조의 장점

폰 노이만 구조를 적용하기 전에는 컴퓨터에 다른 작업을 수행할 경우, 하드웨어 전선을 일일이 재배치하느라 상당한 시간과 인력이 필요했기 때문에 여러 가지 목적의 작업들을 수행하기에는 한계가 있었습니다. 반면, 폰 노이만 구조를 도입한 이후에는 하드웨어는 그대로 두고 소프트웨어(프로그램)만 교체하면 되기 때문에 편의성이 크게 증가하였고 다양한 목적으로 사용이 가능해져 범용성이 향상되었습니다. 이것이 현재 거의 모든 컴퓨터가 폰 노이만 구조를 따르는 특장점입니다.

📄 정리

  • 운영체제의 역할 : 시스템 자원, 응용 프로그램 관리, 효율적 분배
  • 소프트웨어(프로그램)은 운영체제와 응용 프로그램으로 나뉜다.
  • 응용 프로그램은 운영체제 위에서 실행이 된다.
